## 🎯 Q1 功能池（1-3 月）

### 第 1 周：用户认证系统 👤
**优先级**: ⭐⭐⭐⭐⭐ (最高)

**功能描述**:
- 用户注册/登录
- JWT Token 认证
- 密码加密存储
- 用户个人中心

**技术实现**:
- 后端：Spring Security + JWT
- 前端：登录表单 + Token 存储

## 📝 开发规范

### Git 提交规范
```
feat: 新功能
fix: 修复 bug
docs: 文档更新
style: 代码格式
refactor: 重构
test: 测试
chore: 构建/工具
```

### 代码审查
- 所有功能需要 PR 审查
- 至少 1 人 approval
- 通过 CI/CD 检查

### 测试要求
- 单元测试覆盖率 > 70%
- 端到端测试关键流程
- 移动端兼容性测试
